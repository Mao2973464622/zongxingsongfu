# 项目总约束提示词 - 端午安康·灵境

## 1. 项目核心定位
- **主题**：端午节安康（Dragon Boat Festival · Peace & Health）
- **核心情感**：安康、祈福、传承、趣味
- **目标受众**：全年龄段，尤其适合分享给亲友、发朋友圈、家庭群
- **使用场景**：手机端 H5 页面（优先），PC 端自适应（次之）

## 2. 设计要求
- **风格**：新中式 + 轻国潮插画风。避免老气横秋的传统红绿配色，采用「艾草青」「粽叶绿」「糯米白」「朱砂红点缀」为主色调。
- **创意亮点**：
  - 首页必须有动态「赛龙舟」微交互（手指滑动控制龙舟前进/躲避障碍）。
  - 粽子拟人化：点击粽子会弹出不同口味的祝福语（甜/咸/辣/冰）。
  - 隐藏彩蛋：连续点击 5 次页面空白处，触发「艾草飘香」粒子动画 + 随机安康签。
- **动效要求**：
  - 首屏加载动画（龙舟破水而出，持续 2s）。
  - 滚动视差效果（背景山水渐变，前景粽子/艾草浮动）。
  - 按钮 hover/click 反馈（微弹性缩放 + 涟漪扩散）。

## 3. 技术栈与限制
- **前端**：纯 HTML5 + CSS3 + Vanilla JavaScript（无框架依赖，确保轻量快速）。
- **后端**：无后端，纯静态页面。
- **资源**：
  - 所有图片/图标使用 SVG 内联或 base64（减少请求）。
  - 字体：使用 Google Fonts 的 `ZCOOL KuaiLe`（中文） + `Ma Shan Zheng`（书法标题）。
- **性能**：页面总大小 < 500KB，LCP < 1.5s。
- **兼容性**：iOS Safari / Android Chrome / 微信内置浏览器。

## 4. 内容结构（固定 5 屏）
| 屏序 | 名称         | 内容要点                                     |
|------|--------------|----------------------------------------------|
| 1    | 龙舟竞渡     | 动态龙舟游戏入口 + 主标题「端午安康」        |
| 2    | 粽星捧月     | 6 个拟人粽子，点击弹出祝福语                 |
| 3    | 艾草飘香     | 艾草生长动画 + 祈福卡片生成器                |
| 4    | 五色丝线     | 编织五彩绳交互（拖拽连线）+ 许愿             |
| 5    | 安康寄语     | 用户自定义祝福语生成海报（可保存/分享）      |

## 5. 禁止事项
- ❌ 不使用任何外部 JS 库（如 jQuery、GSAP、Three.js）。
- ❌ 不出现真实人物照片或复杂摄影图。
- ❌ 不添加音频自动播放（需用户点击触发）。
- ❌ 不出现任何商业广告或品牌植入。
